## Automatic updates

Cronos AD checks for updates automatically every time it opens. When a new version is available, an update button will appear in the top bar. Click it to download and install the update in one step.

---

## Getting started

1. Open `CronosAD.exe`
2. Follow the **Comece por aqui** steps on the Cadastro tab:
   - Configure your school structure (segments and years)
   - Add class groups
   - Add teachers and their subject assignments
3. Click **Gerar horario** to generate the timetable
4. Export to Excel or image from the **Galeria** tab
